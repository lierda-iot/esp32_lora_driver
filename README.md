# ESP LoRa Driver

[简体中文](README.zh-CN.md) | [English](README.en.md)

This repository provides an ESP-IDF component for Semtech LoRa radio chips, currently focused on `LR20xx / LR2021`.

- Chinese documentation: [README.zh-CN.md](README.zh-CN.md)
- English documentation: [README.en.md](README.en.md)

## Overview

The component keeps Semtech's layered architecture (`RAC`, `RALF`, `RAL`) and adds ESP platform adaptation for SPI, GPIO, timing, interrupts, and logging.
